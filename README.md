# 総トン数別入港数ダッシュボード

## 概要
港湾入港データを総トン数別に可視化するダッシュボードアプリケーションです。
HTMLとJavaScriptのみで構築されており、サーバーやフレームワークは不要です。

## 機能

### データアップロード
- CSVファイルをブラウザから直接アップロード
- ファイル読み込み後、自動的にダッシュボードを表示

### サマリーカード
- 総入港数
- 年度データ数
- 平均年間入港数
- 最高年入港数

### グラフ表示
1. **トン数別入港数の推移** - 折れ線グラフで各トン数カテゴリーの年度別推移を表示
2. **最新年度のトン数別入港数** - 棒グラフで直近年度のトン数別内訳を表示
3. **年度別トン数別入港数の推移（積み上げ）** - 積み上げ棒グラフで全体の推移を表示

### 詳細テーブル
- 全データを表形式で表示
- 検索機能
- ページネーション
- 表示件数変更（10/25/50/100件）

## CSVフォーマット

```csv
年度,500トン以上1000トン未満,1000トン以上5000トン未満,...
2023,157,2051,5921,...
2022,144,2315,6137,...
```

最初の列は「年度」、以降の列は各トン数カテゴリーの入港数となります。

## 使用方法

1. `index.html` をブラウザで開く
2. 「CSVファイルを選択」ボタンをクリック
3. CSVファイル（SJIS形式）を選択
4. 自動的にダッシュボードが表示されます
